# 🚍 OptiBus

Sistema de mobilidade urbana desenvolvido com o objetivo de otimizar rotas e melhorar a experiência de transporte público.

O projeto simula uma solução para gestão de linhas, rotas e informações de transporte, com foco em organização de dados e usabilidade.

---

## 📌 Funcionalidades

- Visualização de rotas e informações de transporte  
- Interface organizada para navegação entre dados  
- Estrutura modular para expansão do sistema  
- Integração entre frontend e backend  

---

## 🛠️ Tecnologias utilizadas

### Frontend
- React  
- TypeScript  
- HTML  
- CSS  

### Backend
- Node.js  
- Express  
- PostgreSQL
